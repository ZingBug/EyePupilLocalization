2017年6月20日增加<br />  
1、修改视频播放机制，不再采用while循环，而是采用定时器方式，每隔20ms读取一次帧数。<br />  
2、与之同时修改的是，视频处理显示时，不会再出现单独窗口了，而是统一在应用程序内。<br />  
3、增加了关闭摄像头按钮，不再采用ESC取消。<br />  
4、增加了眼睛数目变量，用于判断当前处理眼睛数目，假设在缺少一个眼睛视频流情况下，能够自动填充“无视频”标志，但目前不可以，因为那个图像标志有点问题，无法转换后正常显示。<br />  
2017年6月17日增加<br />  
1、添加INI配置文件，将摄像头网络地址保存在配置文件中，以便软件移植时方便修改摄像头地址。<br />  
2、每次软件启动时访问INI配置文件，然后读取保存在其中的网络摄像头地址。<br />  
3、如果想修改网络摄像头地址，不必去修改程序，而只要用记事本修改INI配置文件即可，方便很多。<br />

2017年5月5日增加<br />  
1、采用QT+VS2015+Opencv库方式完成。<br />  
2、与matlab过程相似，处理过程依次为灰度化、二值化、边缘检测、对最大轮廓拟合圆检测（hough圆检测）、定位。<br />  
3、加入了闭眼检测。<br />  
4、对于hough圆检测方法来说，每帧hough圆检测的半径范围都是上一帧的半径加减5得到的。还有每帧hough检测圆心坐标都会与上一帧做判断，如果相距太远，则视为无效数据，相当于上下帧之间有联系。对于拟合圆检测没有这个说法。<br />  
5、给出了两个坐标，数字显示的是绝对坐标，波形显示的是相对坐标，其中坐标原点在视频第一帧定位的瞳孔圆心。<br />  
6、输入为avi格式视频，输出视频在30+帧/秒。<br />
